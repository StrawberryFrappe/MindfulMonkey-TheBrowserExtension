# Mindful Monkey Browser Extension

## Overview
The Mindful Monkey is a browser extension designed to help users cultivate mindfulness in their digital interactions. It provides features that encourage users to take breaks, reflect, and be present while browsing the web.

## Features
- **Break Reminders**: Set customizable reminders to take breaks at intervals you choose.
- **Mindfulness Exercises**: Access guided mindfulness exercises to practice during breaks.
- **Focus Mode**: Disable distracting websites for a set period to help you concentrate.
- **Daily Logs**: Keep track of your mindfulness practices and reflections.

## Installation
1. **Download the Extension**: Get the latest version from our [GitHub releases](https://github.com/StrawberryFrappe/MindfulMonkey-TheBrowserExtension/releases).
2. **Install on Your Browser**: Follow these steps based on your browser:
   - **Chrome**: Go to chrome://extensions/, enable "Developer mode", and click “Load unpacked”. Select the extension folder.
   - **Firefox**: Go to about:debugging#/runtime/this-firefox, click “Load Temporary Add-on”, and select the extension's manifest file.

## Usage
1. Click on the Mindful Monkey icon in your browser toolbar.
2. Configure your settings such as break intervals and focus mode.
3. Start your mindfulness journey while browsing!
