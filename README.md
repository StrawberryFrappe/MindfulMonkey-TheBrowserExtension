# Mindful Monkey

A really simple browser extension that tracks your time on distracting websites and gently redirects you to Wikipedia when you've had enough!

## Features
- **Time Tracking**: Keeps an eye on how long you spend on specific sites.
- **Gentle Nudges**: Replaces your scrolling habits with a redirect to a random Wikipedia article once your limits are exceeded. 

## Installation
1. Download the latest version from our [GitHub releases](https://github.com/StrawberryFrappe/MindfulMonkey-TheBrowserExtension/releases).
2. Unzip the downloaded file.
3. Open Chrome and go to `chrome://extensions/`.
4. Enable **Developer mode** in the top right corner.
5. Click **Load unpacked** and select the unzipped folder.
